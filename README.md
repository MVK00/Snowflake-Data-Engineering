# Snowflake Data Engineering

This repository showcases my structured learning and practical implementation of Snowflake, focusing on cloud data warehousing concepts and scalable data processing.

It covers key features including Snowflake architecture, compute-storage separation, data ingestion techniques, Streams and Tasks for pipeline automation, and advanced functionalities such as Time Travel, Fail-safe, and data cloning.

## Topics Covered
- Snowflake Architecture
- Virtual Warehouses
- Database & Schema Design
- Data Loading (Stages, File Formats)
- Streams (CDC)
- Tasks (Automation)
- Time Travel & Fail Safe
- Cloning

## Hands-on SQL Scripts
All practical implementations are available in the `/sql` folder.

## Key Learnings
- Separation of Compute and Storage
- Multi-cluster shared data architecture
- Cost optimization using virtual warehouses
- Incremental data processing using Streams
- Automation using Tasks
